README!

Grupparbete där vi gjorde en applikation i windows form som är integrerad med en databas.

- Kör Visual Studios som administratör.
- Kör forward engineer när du skapar databasen. Användare läggs till automatisk för cars schemat. 
  Alternativt: Använd ny användare eller ändra manuellt till önskad inloggning via query i sqlConnection, rad 103.
  
- Textbox för år tar endast emot fyra siffror.
